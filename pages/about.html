To create machine learning-ready datasets for materials property prediction, we start from multiple trusted data sources and employ an automated processing pipeline:

We use automated scripts to extract structural information from CIF files, compute derived features to enrich the feature space, integrate multi-source data into unified formats, and convert everything into standardized JSON structures compatible with various machine learning models.

What You'll Find

  • Curated datasets designed for machine learning, such as the H2MOF-ML Dataset and the large-scale hMOF-ML Dataset.
  • Detailed documentation, including Quick Start and Data Format guidance, to get you productive quickly.
  • Download options for both browser-based access and scripted Hugging Face retrieval, ensuring flexible workflows.
  • Helper scripts (e.g., explore.py) and consistent JSON formats so you can load samples via dot notation and integrate them into your ML pipelines.

Step 1: Train your ML model using the H2MOF-ML Dataset (contains labeled GCMC simulation data for 4,034 experimental MOFs)

Step 2: Apply the trained model to the hMOF-ML Dataset to predict hydrogen storage properties for 137,953 hypothetical MOF structures
